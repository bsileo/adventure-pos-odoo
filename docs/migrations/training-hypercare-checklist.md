# Training and hypercare checklist

**Audience:** Shop staff after Dive Shop 360 → Adventure POS go-live.

## Pre go-live training

- [ ] Log in, switch POS if multi-company
- [ ] Open/close **POS session**
- [ ] Look up **customer**; walk-in sale without customer
- [ ] **Barcode** scan vs manual search
- [ ] **Discounts** (if used) and manager approval
- [ ] **Return / refund** flow
- [ ] **Layaway / deposits** — if applicable in Odoo scope (may differ from D360)
- [ ] Basic **inventory lookup** (qty on hand)

## Hypercare window (e.g. first 5–10 business days)

- [ ] Daily check-in with **named implementation contact**
- [ ] **Issue log** — severity, workaround, target fix
- [ ] **Escalation** path for blockers (cannot take payment, wrong tax)
- [ ] End-of-day **reconciliation** habit with accountant notes
